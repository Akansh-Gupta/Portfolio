PROJECT 06 : TextUtils
    TextUtils is a text utility application designed for efficient text manipulation. This project provides a suite of features including versatile case conversion (uppercase, lowercase, title case, sentence case), comprehensive whitespace management (trimming, removal of extra spaces), a quick copy function for streamlined workflows, and detailed text summarization providing character, word, and sentence counts, along with estimated reading time. A live preview dynamically updates as the user types, offering immediate feedback. TextUtils also incorporates a user-friendly dark mode toggle for enhanced visual comfort. This project delivers a practical and accessible tool for everyday text processing needs.
